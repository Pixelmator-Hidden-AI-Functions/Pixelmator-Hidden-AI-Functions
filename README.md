## Overview  
Pixelmator Pro integrates a suite of **ML-powered image processing engines**  
that enhance clarity, detail, tonal accuracy and composition  
through automatic, on-device machine learning models.

These systems operate silently in the background  
and influence how Pixelmator interprets, corrects and reconstructs images.

## Core AI Functions  

### • ML Super Resolution  
Upscales images using deep-learning reconstruction to restore fine details.

### • ML Denoise  
Reduces noise while preserving texture, sharpness and micro-contrast.

### • ML Enhance  
Performs global optimization of exposure, color, highlights, shadows and clarity.

### • ML Match Colors  
Analyzes a reference image and transfers palette, tone curve and luminance style.

### • ML Crop  
Suggests better compositions using object detection and visual balance metrics.
